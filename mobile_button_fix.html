💻 技术实现

1. 安全区域适配

/* 使用CSS安全区域变量 */ .safe-area-button { /* 基础安全距离 + 系统安全区域 */ top: max(20px, calc(env(safe-area-inset-top, 0px) + 20px)) !important; right: max(20px, calc(env(safe-area-inset-right, 0px) + 20px)) !important; }

2. Chrome浏览器特殊处理

/* 针对Chrome移动浏览器的特殊处理 */ @supports (-webkit-touch-callout: none) { .safe-area-button { /* Chrome需要更大的顶部距离 */ top: max(30px, calc(env(safe-area-inset-top, 0px) + 30px)) !important; } }

3. 按钮样式增强

.image-modal button { width: 48px; /* 增大尺寸 */ height: 48px; background-color: rgba(0, 0, 0, 0.8) !important; /* 增强背景 */ border: 2px solid rgba(255, 255, 255, 0.3); /* 添加边框 */ }

4. 整体弹窗安全区域

.image-modal { /* 确保整个弹窗在安全区域内 */ padding-top: env(safe-area-inset-top, 0); padding-bottom: env(safe-area-inset-bottom, 0); padding-left: env(safe-area-inset-left, 0); padding-right: env(safe-area-inset-right, 0); }
